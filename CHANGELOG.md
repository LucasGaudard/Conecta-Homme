# Changelog

## V1.1 - Release Premium

Principais melhorias:

- Polimento visual completo da interface.
- Responsividade para desktop, tablet e mobile.
- Sidebar responsiva.
- UX premium com skeletons, loading states, toasts e microinteracoes.
- Componentes reutilizaveis de carregamento e submit.
- Tabelas inteligentes com paginacao, ordenacao, busca debounce e filtros.
- Dashboards premium para Admin, Portaria e Morador com dados reais do Prisma.
- Central de notificacoes persistidas.
- Sino de notificacoes no Header.
- Configuracoes da conta para usuario logado.
- Configuracoes administrativas do condominio.
- Auditoria de acoes importantes.
- Exportacao CSV para relatorios, encomendas e auditoria.
- Melhorias de acessibilidade e navegacao por teclado.
- Documentacao revisada para release e portfolio.

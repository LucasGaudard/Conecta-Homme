# Conecta Homme

Sistema web para gestão de condomínios desenvolvido com foco em segurança, organização e experiência do usuário.

## Funcionalidades

- Gestão de moradores
- Gestão de unidades
- Controle de visitantes
- Controle de acesso
- QR Code para moradores e visitantes
- Controle de encomendas
- Dashboard administrativo
- Dashboard da portaria
- Área exclusiva do morador
- Relatórios e indicadores

## Tecnologias

- Next.js
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- React
- Recharts

## Diferenciais

- Interface moderna e responsiva
- Controle de acesso por perfis
- Autenticação segura
- Dashboard com indicadores
- QR Code para liberação de acesso
- Arquitetura escalável
- Código organizado e componentizado
